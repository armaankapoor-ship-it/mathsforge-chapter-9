01Basics

Differential equation meaning

It contains an unknown function and its derivative or differentials.

  • CBSE writing: Identify dependent and independent variables.
  • JEE focus: Classify before solving.
  • Common trap: An ordinary equation without derivative is not a differential equation.
02Order

Order

Order is the order of the highest derivative present.

  • CBSE writing: Look for highest d^n y/dx^n.
  • JEE focus: Order is usually quick to identify.
  • Common trap: Do not confuse order with power.
03Degree

Degree

Degree is the power of the highest order derivative after making the equation polynomial in derivatives.

  • CBSE writing: Remove radicals and fractions in derivatives before degree.
  • JEE focus: If not polynomial in derivatives, degree is not defined.
  • Common trap: Degree is not always defined.
04Formation

Forming differential equation

Differentiate as many times as there are arbitrary constants, then eliminate them.

  • CBSE writing: Show elimination of constants.
  • JEE focus: Number of constants usually equals order.
  • Common trap: Do not eliminate variables instead of constants.
05Separable

Variable separable equations

Write f(y)dy = g(x)dx and integrate both sides.

  • CBSE writing: Keep y terms and x terms separated.
  • JEE focus: Initial conditions determine constant.
  • Common trap: Do not integrate before separating correctly.
06Homogeneous

Homogeneous equations

If dy/dx is a function of y/x, use y=vx.

  • CBSE writing: Set y=vx and dy/dx=v+x dv/dx.
  • JEE focus: This converts into separable form.
  • Common trap: Derivative of vx is v + x dv/dx.
07Linear

Linear differential equation

dy/dx + P(x)y = Q(x) has solution using integrating factor.

  • CBSE writing: Write standard form before finding IF.
  • JEE focus: Correct P(x) is everything multiplying y.
  • Common trap: Wrong standard form gives wrong IF.
08Solution

General and particular solution

General solution contains arbitrary constant; particular solution uses given condition.

  • CBSE writing: Use condition after general solution.
  • JEE focus: Check solution by differentiating.
  • Common trap: Do not substitute initial condition too early.

Formula sheet

High-yield formulas with exact usage conditions.

Use these as quick revision cards before solving examples.

Order

Order

Order = highest derivative order present

d2y/dx2 gives order 2.

  • Example: y prime + y = 0 has order 1.
  • Trap: Not the exponent.
Degree

Degree

Degree = power of highest order derivative when polynomial in derivatives

Only defined after clearing radicals/fractions of derivatives.

  • Example: (d2y/dx2)^3 + y = 0 has degree 3.
  • Trap: May be undefined.
Separable

Variable separable

dy/dx = g(x)h(y) => dy/h(y) = g(x)dx

Separate variables first.

  • Example: dy/dx=xy gives dy/y=x dx.
  • Trap: Do not mix variables.
Linear

Linear DE standard form

dy/dx + P(x)y = Q(x)

First order linear equation.

  • Example: dy/dx + 2y = e^x.
  • Trap: Coefficient of dy/dx must be 1.
IF

Integrating factor

IF = e^(integral P dx)

Multiplier for linear DE.

  • Example: For P=2, IF=e^(2x).
  • Trap: Use P after standard form.
Solution

Linear solution

y(IF) = integral Q(IF) dx + C

Solution after multiplying by IF.

  • Example: Use for dy/dx + Py = Q.
  • Trap: Do not forget constant.
Homogeneous

Homogeneous substitution

y=vx, dy/dx = v + x dv/dx

Converts function of y/x into separable form.

  • Example: dy/dx=(x+y)/x = 1+y/x.
  • Trap: Derivative of v is not zero.
Formation

Formation rule

Number of differentiations = number of arbitrary constants

Eliminate constants from family.

  • Example: y=mx+c needs two differentiations.
  • Trap: Constants, not variables, are eliminated.

Derivation bank

Proofs that can be reproduced in exams.

Open a proof, follow the steps, and check the final result.

Solve variable separable form
  1. Start with dy/dx=g(x)h(y).
  2. Divide by h(y).
  3. Multiply by dx.
  4. Integrate both sides and add constant.

Final: integral dy/h(y)=integral g(x)dx + C.

Shortcut: Move variables to their sides.

Warning: Division by h(y) assumes nonzero cases separately.

Derive linear equation solution
  1. Start with dy/dx+Py=Q.
  2. Multiply by IF.
  3. Choose IF so IF prime = P IF.
  4. Left side becomes d(y IF)/dx, then integrate.

Final: y(IF)=integral Q(IF)dx + C.

Shortcut: Choose IF so left side becomes product derivative.

Warning: Equation must be in standard form.

Homogeneous substitution
  1. Let y=vx.
  2. Differentiate with respect to x.
  3. dy/dx = v dx/dx + x dv/dx.
  4. So dy/dx = v + x dv/dx.

Final: dy/dx = v + x dv/dx.

Shortcut: Differentiate y=vx.

Warning: v is a function of x.

Formation of DE from family
  1. Count arbitrary constants in the family.
  2. Differentiate that many times.
  3. Use original and derivative equations.
  4. Eliminate constants to get the differential equation.

Final: Eliminate arbitrary constants.

Shortcut: Differentiate enough times.

Warning: Do not eliminate x or y.

Practice bank

Exam-style questions with answer reveal.

Filter by keyword, try the question first, then reveal the answer and trap.

OrderEasy

1. Find order of d2y/dx2 + y = 0.

Answer: 2

Highest derivative is second derivative.

Trap: Do not use power of y.

DegreeMedium

2. Find degree of (d2y/dx2)^3 + y = 0.

Answer: 3

Highest derivative is second order and its power is 3.

Trap: Degree is not order.

DegreeHard

3. Is degree defined for sqrt(dy/dx)+y=0?

Answer: Not in given form

Degree is defined only when equation is polynomial in derivatives.

Trap: Do not call degree 1/2.

SeparableMedium

4. Solve dy/dx=xy.

Answer: log|y|=x^2/2+C

dy/y=x dx, integrate.

Trap: Separate before integrating.

LinearMedium

5. What is IF for dy/dx+2y=e^x?

Answer: e^(2x)

P=2, so IF=e^(integral 2 dx).

Trap: Use P from standard form.

HomogeneousMedium

6. For y=vx, what is dy/dx?

Answer: v+x dv/dx

Use product rule.

Trap: v is a function of x.

FormationMedium

7. How many times differentiate y=mx+c to form DE?

Answer: 2 times

There are two arbitrary constants m and c.

Trap: Constants decide differentiations.

SolutionEasy

8. What is a general solution?

Answer: Solution containing arbitrary constant

It represents a family of solution curves.

Trap: Particular solution uses a condition.

LinearHard

9. Write solution formula for dy/dx+Py=Q.

Answer: y IF = integral Q IF dx + C

IF=e^(integral P dx).

Trap: Equation must be standard first.

CBSEMedium

10. Classify dy/dx = x+y.

Answer: First order differential equation

Highest derivative is first derivative.

Trap: It is not variable separable directly.

JEEHard

11. If dy/dx=(x+y)/x, what substitution helps?

Answer: y=vx

Right side is 1+y/x, a function of y/x.

Trap: Use homogeneous substitution.

TrapMedium

12. Can you apply IF before standard form?

Answer: No

First make coefficient of dy/dx equal to 1.

Trap: Wrong P gives wrong IF.

Core formulas

  • Order: Order = highest derivative order present
  • Degree: Degree = power of highest order derivative when polynomial in derivatives
  • Variable separable: dy/dx = g(x)h(y) => dy/h(y) = g(x)dx
  • Linear DE standard form: dy/dx + P(x)y = Q(x)
  • Integrating factor: IF = e^(integral P dx)
  • Linear solution: y(IF) = integral Q(IF) dx + C

Common traps

  • Degree is defined only for polynomial equations in derivatives.
  • Number of arbitrary constants guides formation.
  • Separable means variables must be separated first.
  • In y=vx, dy/dx=v+x dv/dx.
  • Linear equation must be in standard form.
  • General solution includes arbitrary constant.
